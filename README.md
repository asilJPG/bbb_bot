# 🤖 Telegram Bot — Инструкция по запуску

## 📁 Структура проекта

```
tgbot/
├── bot.py           # Точка входа
├── config.py        # Конфигурация (токен, ID)
├── database.py      # Работа с SQLite
├── keyboards.py     # Клавиатуры
├── states.py        # FSM-состояния
├── handlers/
│   ├── __init__.py
│   ├── user.py      # Обработчики пользователей
│   └── admin.py     # Обработчики администратора
├── requirements.txt
└── README.md
```

---

## ⚙️ 1. Установка

```bash
# Создайте виртуальное окружение (рекомендуется)
python -m venv venv
source venv/bin/activate        # Linux/Mac
venv\Scripts\activate           # Windows

# Установите зависимости
pip install -r requirements.txt
```

---

## 🔑 2. Настройка config.py

Откройте файл `config.py` и заполните три значения:

```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"   # Токен от @BotFather
ADMIN_ID  = 123456789               # Ваш Telegram user_id
CHAT_ID   = -1001234567890          # ID чата/канала для заявок
```

### Как получить значения:

| Значение | Как получить |
|----------|-------------|
| `BOT_TOKEN` | Создайте бота у @BotFather → `/newbot` |
| `ADMIN_ID` | Напишите @userinfobot или @getidsbot |
| `CHAT_ID` | Добавьте бота в канал/группу как администратора, отправьте сообщение, используйте @getidsbot |

---

## ▶️ 3. Запуск

```bash
python bot.py
```

При первом запуске автоматически создаётся файл `bot.db` (SQLite).

---

## 🔐 4. Команды администратора

| Команда | Описание |
|---------|----------|
| `/admin` | Открыть панель управления |
| `/broadcast` | Начать рассылку |

В панели администратора можно изменить:
- Видео 1 и Видео 2 (file_id из Telegram или ссылка)
- Описание к видео 1 и 2
- YouTube-ссылку для реферального сценария

---

## 👤 5. Пользовательские сценарии

### Сценарий 1 — обычный старт:
```
/start
→ Бот отправляет 2 видео
→ Через 5 минут спрашивает имя
→ Затем просит номер телефона
→ Отправляет заявку в CHAT_ID
```

### Сценарий 2 — реферальная ссылка:
```
https://t.me/YOUR_BOT?start=video
→ Бот отправляет YouTube-ссылку
```

---

## 🎬 6. Как добавить видео

**Вариант A — через file_id:**
1. Отправьте видео напрямую боту
2. Используйте @getidsbot или свой код для получения `file_id`
3. Вставьте `file_id` через `/admin` → "Изменить видео"

**Вариант B — ссылка:**
- Вставьте прямую URL-ссылку на видео (например, из Telegram CDN)

---

## 📝 Примечания

- База данных `bot.db` создаётся автоматически рядом с `bot.py`
- Логи пишутся в `bot.log` и в консоль
- Если пользователь уже ввёл имя и телефон — при повторном `/start` данные не перезапрашиваются
- Если пользователь пишет что-то в течение 5-минутного таймера — диалог начинается досрочно
